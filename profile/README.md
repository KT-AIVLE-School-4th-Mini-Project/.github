# 📚 모두의 서재

누구나 작가가 되어 자유롭게 글을 집필하고 공개할 수 있는 창작 플랫폼

> '모두의 서재' GitHub Organization에 오신 것을 환영합니다! <br>
AI 표지 생성 기능과 직관적인 독서/집필 경험을 통해 창작의 장벽을 낮추는 서비스를 만들고 있습니다.

## 📂 Repositories

프로젝트는 크게 프론트엔드와 백엔드 두 개의 메인 리포지토리로 구성되어 있습니다.

| 영역 | 저장소 (repository) | 주요 기술 스택 |
| --- | --- | --- |
| Frontend | FrontEnd | React, Vite, MUI |
| Backend | BackEnd | Java 17, Spring Boot 3.5.8, JPA, H2 Database |

## 🚀 Key Features

### 1. AI 표지 생성 (AI Cover Generation)

작가가 글을 등록할 때, 본문의 분위기에 맞는 표지를 AI가 3장 제안합니다.

창작자는 가장 마음에 드는 이미지를 선택하여 작품을 완성할 수 있습니다.

### 2. 작가와 독자를 잇는 공간

작가: 쉽고 편안한 에디터로 글을 쓰고 관리합니다.

독자: 관심 도서를 등록하고, 작가의 새로운 이야기를 가장 먼저 만나봅니다.
